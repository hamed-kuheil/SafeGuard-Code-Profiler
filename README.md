> **SafeGuard** is a smart & lightweight tool for Computer Engineering students. It scans C++, Java, and Python code for performance, security, and privacy issues — 100% offline.

---

## 🧠 Engineering Philosophy | فلسفة المشروع

بصفتي ادرس هندسة الحاسوب، أؤمن أن الكود ليس مجرد منطق، بل هو مسؤولية تجاه العتاد والمستخدم:

* **⚡ Hardware Awareness (احترام العتاد):** الكود السيئ يرهق المعالج. نحن نبحث عن "الكود الأخضر" الذي يوفر الطاقة.
* **🔒 Security as Correctness (الأمان أولاً):** الأمان ليس رفاهية؛ الكود الذي يسرب البيانات هو كود خاطئ هندسياً.
* **🧭 Ethical Compliance (الجانب الإنساني):** حماية خصوصية البشر هي جوهر تطويرنا في مشروع **SafeGuard-AI**.

---

## 🛠️ Key Features | أهم المميزات

* **Efficiency Check:** Finds nested loops and memory leaks in C++.
* **Security Audit:** Detects hardcoded passwords and unsafe functions.
* **Privacy Scanner:** Flags PII (Personal Information) like emails and IDs.
* **Modern Reports:** Generates cool HTML dashboards and JSON files.

---

## 🚀 Quick Start | ابدأ الآن

**1. Install Dependencies:**

```bash
pip install colorama

```

**2. Scan a file:**

```bash
python src/main.py path/to/your/code.cpp

```

**3. Generate HTML Report:**

```bash
python src/main.py examples/ --html

```

## 📊 The Scoring System | نظام التقييم

يعطيك البرنامج تقييماً من 10 بناءً على جودة الكود الهندسي:

* **Efficiency Score:** يقيس كفاءة استهلاك الموارد (CPU/RAM).
* **Ethical Score:** يقيس مدى أمان وخصوصية بيانات المستخدم.
